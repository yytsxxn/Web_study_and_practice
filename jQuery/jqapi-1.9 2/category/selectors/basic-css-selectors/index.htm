分类：基础

下面的选择是基于层叠样式表1规范，  作为W3C概述。 有关规格的更多信息， 参观http://www.w3.org/Style/CSS/#specs。

All Selector (“*”)

选择所有元素.

Class Selector (“.class”)

选择给定样式类名的所有元素。

Element Selector (“element”)

根据给定（html）标记名称选择所有的元素。

ID Selector (“#id”)

选择一个具有给定id属性的单个元素。

Multiple Selector (“selector1, selector2, selectorN”)

将每一个选择器匹配到的元素合并后一起返回。
